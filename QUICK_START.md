## Troubleshooting

**App won't load?**
- Check file path correct
- All files in same folder?

**Tasks not saving?**
- Browser allows localStorage?
- Try Chrome/Edge?

**Weekend logic off?**
- Is your device date correct?
- Try refresh page

**Voice not working?**
- Using Chrome/Edge?
- Microphone connected?
- Granted permission?
